A modern Weather Web Application built using PHP, MySQL, JavaScript, and OpenWeather API.
This project provides real-time weather updates, forecasts, charts, and a user authentication system.

📌 About the Project
WeatherPulse provides real-time weather data, forecasts, and intelligent suggestions based on environmental conditions.

This project demonstrates integration of:

Frontend UI/UX design
Backend authentication system
External API consumption
Data visualization using charts
